# Oboe_vs_auto-sklearn
Experiment bench to compare performance of Oboe to auto-sklearn


### Installation
* Clone this repository into a directory of your choice.
* Clone the Oboe repository into a directory of your choice (such as at the same level as this one).
* Satisfy the package dependencies. You need at least Python 3.6. This can easily be done with:

    `pip install openml auto-sklearn`

* Prepare your `experiment_config.py` file. You can use the `config_example.py` file as a starting point.
